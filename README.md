# moods

Static exports of the How We Feel Android mood assets.

Contents:

- `index.html`: static catalogue page linking the exported files
- `data/catalog.yaml`: mood ids used for the catalogue
- `svg/`: 322 exported SVG assets
- `png/`: 322 exported PNG assets
- `manifest.tsv`: source-to-export mapping from the APK extraction step

Source app:

- site: `https://howwefeel.org/`
- package: `org.howwefeel.moodmeter`

These files were extracted from the Android APK for reference and inspection.
